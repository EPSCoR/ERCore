INTRODUCTION
------------

This ERCore Dashboard module displays team content and my incomplete/complete content on user profile pgae.
This module modifes the custom module written by Amy Walsh at 
https://github.com/EPSCoR/ERCore-3.1/tree/3.1-dev-teamview2 [er_addteamview]

INSTALLATION
------------
 
 * Install EPSCoR User Dashboard 7.3.x (styles and views gets added automatically).
 
 * You may need to disable 'EPSCoR Add Team View' module otherwise a duplicate view will be shown
